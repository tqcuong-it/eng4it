# 50 Bài Từ Vựng IT — Content Plan

## Format mỗi bài:
- 10-20 từ vựng
- Bảng: English | IPA | Tiếng Việt | Example
- Chia nhóm nhỏ theo chủ đề con
- Bài tập: Fill in the blank + Matching
- Đáp án (spoiler)

## Batch 1: AWS Services (1-10)
1. AWS EC2 — Thuật ngữ máy chủ ảo (instance, AMI, security group, elastic IP...)
2. AWS S3 — Thuật ngữ lưu trữ đám mây (bucket, object, lifecycle, versioning...)
3. AWS Lambda — Thuật ngữ Serverless (function, trigger, cold start, runtime...)
4. AWS RDS — Thuật ngữ cơ sở dữ liệu đám mây (replica, snapshot, failover, endpoint...)
5. AWS VPC — Thuật ngữ mạng đám mây (subnet, route table, NAT gateway, CIDR...)
6. AWS IAM — Thuật ngữ bảo mật & phân quyền (policy, role, MFA, principal...)
7. AWS CloudFormation — Thuật ngữ Infrastructure as Code (stack, template, drift, rollback...)
8. AWS ECS & EKS — Thuật ngữ Container & Kubernetes (cluster, task, pod, service...)
9. AWS CloudWatch — Thuật ngữ Monitoring & Logging (metric, alarm, dashboard, log group...)
10. AWS Route 53 & API Gateway — Thuật ngữ DNS & API (hosted zone, record, throttling, stage...)

## Batch 2: AWS + Cloud (11-15)
11. AWS SQS & SNS — Thuật ngữ Message Queue (queue, topic, subscriber, dead-letter...)
12. AWS DynamoDB — Thuật ngữ NoSQL Database (partition key, GSI, throughput, stream...)
13. AWS CodePipeline — Thuật ngữ CI/CD trên AWS (source, build, deploy, artifact...)
14. Cloud Computing Cơ Bản — IaaS, PaaS, SaaS, multi-tenant, scalability...
15. AWS Cost Optimization — Thuật ngữ tối ưu chi phí (reserved, spot, savings plan, billing...)

## Batch 3: DevOps & Tools (16-25)
16. Docker — Thuật ngữ Container (image, container, volume, network, Dockerfile...)
17. Git — Thuật ngữ Version Control (commit, branch, merge, rebase, cherry-pick...)
18. CI/CD Pipeline — Thuật ngữ tích hợp & triển khai liên tục (build, test, deploy, artifact, rollback...)
19. Kubernetes — Thuật ngữ K8s (pod, deployment, service, ingress, configmap, namespace...)
20. Linux & Terminal — Thuật ngữ dòng lệnh (shell, pipe, permission, process, daemon...)
21. Networking — Thuật ngữ mạng (TCP/IP, DNS, HTTP, firewall, load balancer, proxy...)
22. Database Fundamentals — Thuật ngữ cơ sở dữ liệu (schema, index, query, transaction, normalization...)
23. API & REST — Thuật ngữ giao tiếp hệ thống (endpoint, request, response, middleware, payload...)
24. Security & Cybersecurity — Thuật ngữ an ninh mạng (encryption, authentication, vulnerability, firewall...)
25. Monitoring & Observability — Thuật ngữ giám sát hệ thống (metrics, tracing, alerting, SLA, uptime...)

## Batch 4: Development (26-35)
26. Frontend Development — Thuật ngữ lập trình giao diện (component, state, props, render, DOM...)
27. Backend Development — Thuật ngữ lập trình server (API, middleware, ORM, migration, caching...)
28. Testing & QA — Thuật ngữ kiểm thử (unit test, integration test, coverage, mock, assertion...)
29. Microservices Architecture — Thuật ngữ kiến trúc (service mesh, circuit breaker, saga, gateway...)
30. System Design — Thuật ngữ thiết kế hệ thống (scalability, availability, consistency, partition...)
31. Machine Learning Cơ Bản — Thuật ngữ AI/ML (model, training, inference, dataset, accuracy...)
32. Mobile Development — Thuật ngữ lập trình di động (native, hybrid, responsive, push notification...)
33. Data Engineering — Thuật ngữ xử lý dữ liệu (ETL, pipeline, data lake, warehouse, batch...)
34. Code Review — Thuật ngữ review code (pull request, approval, comment, refactor, suggestion...)
35. Technical Debt — Thuật ngữ nợ kỹ thuật (legacy, refactor, deprecate, migration, workaround...)

## Batch 5: Workplace & Communication (36-50)
36. Daily Standup — Từ vựng họp hàng ngày (blocker, progress, yesterday, today, impediment...)
37. Sprint Planning — Từ vựng lập kế hoạch sprint (backlog, story point, velocity, acceptance criteria...)
38. Agile & Scrum — Từ vựng quản lý dự án Agile (sprint, retrospective, kanban, user story...)
39. Technical Interview — Từ vựng phỏng vấn kỹ thuật (whiteboard, algorithm, complexity, trade-off...)
40. Email Writing — Từ vựng viết email chuyên nghiệp (regarding, attached, follow up, deadline...)
41. Bug Report — Từ vựng báo cáo lỗi (reproduce, severity, priority, workaround, root cause...)
42. Project Kickoff — Từ vựng khởi động dự án (scope, milestone, stakeholder, timeline, deliverable...)
43. Client Communication — Từ vựng giao tiếp khách hàng (requirement, clarification, feedback, approval...)
44. Performance Review — Từ vựng đánh giá hiệu suất (KPI, objective, achievement, improvement, feedback...)
45. Onboarding — Từ vựng nhập môn nhân viên mới (orientation, handbook, mentor, probation, policy...)
46. Salary & Benefits — Từ vựng lương thưởng (compensation, bonus, equity, benefits, negotiation...)
47. Remote Work — Từ vựng làm việc từ xa (asynchronous, timezone, screen sharing, bandwidth...)
48. Team Collaboration — Từ vựng làm việc nhóm (brainstorm, delegate, escalate, consensus, alignment...)
49. Technical Documentation — Từ vựng viết tài liệu (README, changelog, specification, diagram, reference...)
50. Release & Deployment — Từ vựng phát hành phần mềm (release notes, hotfix, patch, rollback, downtime...)
