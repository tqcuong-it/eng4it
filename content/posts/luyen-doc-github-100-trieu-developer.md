## 📰 Bài đọc (English)

GitHub, which is the world's largest code {{< vocab w="hosting" ipa="/ˈhoʊstɪŋ/" vn="lưu trữ" t="noun" >}} platform, has reached 100 million {{< vocab w="registered" ipa="/ˈredʒɪstərd/" vn="đã đăng ký" t="adj" >}} developers. The {{< vocab w="milestone" ipa="/ˈmaɪlstoʊn/" vn="cột mốc" t="noun" >}} shows how fast the global developer {{< vocab w="community" ipa="/kəˈmjuːnəti/" vn="cộng đồng" t="noun" >}} is growing.

## 🇻🇳 Bản dịch tiếng Việt

GitHub, nền tảng lưu trữ mã nguồn lớn nhất thế giới, đã đạt mốc 100 triệu lập trình viên đăng ký. Cột mốc này cho thấy cộng đồng developer toàn cầu đang phát triển nhanh chóng.

Microsoft, mua lại GitHub vào năm 2018 với giá 7,5 tỷ đô la, đã đầu tư mạnh vào các tính năng mới. GitHub Copilot, công cụ AI giúp lập trình viên viết code, đã trở thành một trong những tính năng phổ biến nhất. Nếu lập trình viên sử dụng Copilot, họ sẽ hoàn thành công việc nhanh hơn tới 55%.

Nền tảng hiện lưu trữ hơn 330 triệu repository. Các lập trình viên đóng góp cho các dự án mã nguồn mở cho biết GitHub giúp việc cộng tác dễ dàng hơn bao giờ hết. Nếu tốc độ tăng trưởng tiếp tục như hiện tại, GitHub sẽ đạt 200 triệu người dùng vào năm 2028. Đa số người dùng mới đến từ Ấn Độ, Brazil và Đông Nam Á.

## 📝 Phân tích ngữ pháp

### Câu 1: "GitHub, which is the world's largest code hosting platform, has reached 100 million registered developers."
- **Cấu trúc:** S + , which + is + N , + has V-ed — mệnh đề quan hệ không xác định
- **Ngữ pháp:** Non-defining relative clause giới thiệu GitHub
- **Ví dụ tương tự:** Stack Overflow, which is a Q&A site for developers, has millions of users.

### Câu 2: "GitHub Copilot, an AI tool that helps developers write code, has become one of the most popular features."
- **Cấu trúc:** S + , appositive + that + V , + has become — đồng vị ngữ + mệnh đề quan hệ
- **Ngữ pháp:** Appositive phrase kết hợp defining relative clause
- **Ví dụ tương tự:** ChatGPT, an AI chatbot that understands natural language, has changed how people search.

### Câu 3: "If developers use Copilot, they will complete tasks up to 55% faster."
- **Cấu trúc:** If + S + V (present), S + will + V — câu điều kiện loại 1
- **Ngữ pháp:** First conditional diễn tả lợi ích thực tế
- **Ví dụ tương tự:** If you use a linter, you will catch errors before running the code.

### Câu 4: "Developers who contribute to open-source projects say GitHub makes collaboration easier than ever."
- **Cấu trúc:** S + who + V + V + that + S + V — mệnh đề quan hệ xác định
- **Ngữ pháp:** Defining relative clause xác định nhóm developer + so sánh hơn
- **Ví dụ tương tự:** Engineers who work in agile teams say daily standups improve communication.

2. Microsoft ___ GitHub in 2018 for $7.5 billion.
3. Many developers ___ to open-source projects in their free time.
4. Good tools make team ___ much easier.
