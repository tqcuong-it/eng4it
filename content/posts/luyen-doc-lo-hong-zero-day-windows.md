> ⏱️ **Thời gian đọc:** ~4 phút
> 📰 **Chủ đề:** Zero-Day Vulnerability

## 📰 Bài đọc (English)

Microsoft has released an {{< vocab w="emergency patch" ipa="/ɪˈmɜːrʤənsi pætʃ/" vn="bản vá khẩn cấp" t="noun" >}} for a {{< vocab w="zero-day vulnerability" ipa="/ˈzɪroʊ deɪ ˌvʌlnərəˈbɪləti/" vn="lỗ hổng zero-day" t="noun" >}} that was found in Windows. The {{< vocab w="flaw" ipa="/flɔː/" vn="lỗ hổng, khiếm khuyết" t="noun" >}}, which affects millions of devices, allows {{< vocab w="hackers" ipa="/ˈhækərz/" vn="tin tặc" t="noun" >}} to run {{< vocab w="malicious" ipa="/məˈlɪʃəs/" vn="độc hại" t="adj" >}} code without the user's knowledge.

Security researchers who discovered the bug say that {{< vocab w="attackers" ipa="/əˈtækərz/" vn="kẻ tấn công" t="noun" >}} have already {{< vocab w="exploited" ipa="/ɪkˈsplɔɪtɪd/" vn="khai thác (lỗ hổng)" t="verb" >}} it in real attacks. If users do not {{< vocab w="update" ipa="/ʌpˈdeɪt/" vn="cập nhật" t="verb" >}} their systems immediately, their computers will remain {{< vocab w="vulnerable" ipa="/ˈvʌlnərəbəl/" vn="dễ bị tấn công" t="adj" >}}.

Microsoft strongly recommends that all Windows users install the {{< vocab w="security update" ipa="/sɪˈkjʊrəti ʌpˈdeɪt/" vn="bản cập nhật bảo mật" t="noun" >}} as soon as possible. Companies that manage large networks should {{< vocab w="prioritize" ipa="/praɪˈɔːrɪtaɪz/" vn="ưu tiên" t="verb" >}} this patch to protect their systems from further damage.

## 🇻🇳 Bản dịch tiếng Việt

Microsoft đã phát hành bản vá khẩn cấp cho một lỗ hổng zero-day được phát hiện trong Windows. Lỗ hổng này ảnh hưởng đến hàng triệu thiết bị, cho phép tin tặc chạy mã độc hại mà người dùng không hay biết.

Các nhà nghiên cứu bảo mật phát hiện lỗi cho biết kẻ tấn công đã khai thác nó trong các cuộc tấn công thực tế. Nếu người dùng không cập nhật hệ thống ngay lập tức, máy tính của họ sẽ vẫn dễ bị tấn công.

Microsoft khuyến nghị mạnh mẽ tất cả người dùng Windows cài đặt bản cập nhật bảo mật càng sớm càng tốt. Các công ty quản lý mạng lớn nên ưu tiên bản vá này để bảo vệ hệ thống khỏi thiệt hại thêm.

## 📝 Phân tích ngữ pháp

### Câu 1: "The flaw, which affects millions of devices, allows hackers to run malicious code."
- **Cấu trúc:** S + , which + V + O + , + V + O + to V + O
- **Ngữ pháp:** Mệnh đề quan hệ không xác định (Non-defining relative clause) với "which" — bổ sung thông tin về lỗ hổng
- **Ví dụ tương tự:** The bug, which was reported last week, causes the app to crash.

### Câu 2: "Security researchers who discovered the bug say that attackers have already exploited it."
- **Cấu trúc:** S + who + V + O + V + that + S + have already + V(past participle) + O
- **Ngữ pháp:** Mệnh đề quan hệ xác định với "who" + câu tường thuật với "that"
- **Ví dụ tương tự:** Developers who tested the feature say that it works well.

### Câu 3: "If users do not update their systems immediately, their computers will remain vulnerable."
- **Cấu trúc:** If + S + do not + V + O, S + will + remain + adj
- **Ngữ pháp:** Câu điều kiện loại 1 phủ định (Negative first conditional) — cảnh báo hậu quả thực tế
- **Ví dụ tương tự:** If you do not back up your data, you will lose everything.

### Câu 4: "Companies that manage large networks should prioritize this patch."
- **Cấu trúc:** S + that + V + O + should + V + O
- **Ngữ pháp:** Mệnh đề quan hệ xác định với "that" + "should" diễn tả lời khuyên
- **Ví dụ tương tự:** Teams that work with sensitive data should use encryption.

## ✏️ Bài tập

### Comprehension (Đọc hiểu)
1. Lỗ hổng zero-day cho phép tin tặc làm gì?
2. Kẻ tấn công đã làm gì với lỗ hổng này?
3. Microsoft khuyến nghị người dùng làm gì?

### Vocabulary (Từ vựng)
Điền từ thích hợp:
1. Hackers ___ the vulnerability before Microsoft could fix it.
2. The system is ___ to attacks if you don't install the patch.
3. Microsoft released an ___ ___ to fix the critical bug.
4. Companies should ___ security updates over new features.

{{< spoiler "✅ Đáp án" >}}
**Comprehension:**
1. Cho phép tin tặc chạy mã độc hại (malicious code) mà người dùng không hay biết.
2. Kẻ tấn công đã khai thác (exploited) lỗ hổng này trong các cuộc tấn công thực tế.
3. Cài đặt bản cập nhật bảo mật (security update) càng sớm càng tốt.

**Vocabulary:**
1. **exploited** — khai thác
2. **vulnerable** — dễ bị tấn công
3. **emergency patch** — bản vá khẩn cấp
4. **prioritize** — ưu tiên
