## 📰 Bài đọc (English)

TypeScript, which is a {{< vocab w="superset" ipa="/ˈsuːpərˌset/" vn="tập cha, bao gồm" t="noun" >}} of JavaScript, has become one of the most popular languages on GitHub. According to a recent {{< vocab w="survey" ipa="/ˈsɜːrveɪ/" vn="khảo sát" t="noun" >}}, TypeScript now {{< vocab w="ranks" ipa="/ræŋks/" vn="xếp hạng" t="verb" >}} higher than JavaScript in new {{< vocab w="repositories" ipa="/rɪˈpɑːzɪˌtɔːriz/" vn="kho mã nguồn" t="noun" >}} created this year.

Developers who work on large projects prefer TypeScript because it offers {{< vocab w="static typing" ipa="/ˈstætɪk ˈtaɪpɪŋ/" vn="kiểu dữ liệu tĩnh" t="noun" >}}. If a developer makes a {{< vocab w="type error" ipa="/taɪp ˈerər/" vn="lỗi kiểu dữ liệu" t="noun" >}}, the {{< vocab w="compiler" ipa="/kəmˈpaɪlər/" vn="trình biên dịch" t="noun" >}} will catch it before the code runs. This feature helps teams {{< vocab w="reduce" ipa="/rɪˈduːs/" vn="giảm" t="verb" >}} bugs in {{< vocab w="production" ipa="/prəˈdʌkʃən/" vn="môi trường sản phẩm" t="noun" >}}.

Many companies that build web applications have already {{< vocab w="migrated" ipa="/maɪˈɡreɪtɪd/" vn="chuyển đổi" t="verb" >}} their codebases to TypeScript. If this trend continues, TypeScript could {{< vocab w="dominate" ipa="/ˈdɑːmɪneɪt/" vn="thống trị" t="verb" >}} web development within the next few years.

## 🇻🇳 Bản dịch tiếng Việt

TypeScript, vốn là một tập cha của JavaScript, đã trở thành một trong những ngôn ngữ phổ biến nhất trên GitHub. Theo một khảo sát gần đây, TypeScript hiện xếp hạng cao hơn JavaScript về số kho mã nguồn mới được tạo trong năm nay.

Các lập trình viên làm việc trên các dự án lớn ưa chuộng TypeScript vì nó cung cấp kiểu dữ liệu tĩnh. Nếu lập trình viên mắc lỗi kiểu dữ liệu, trình biên dịch sẽ phát hiện trước khi mã chạy. Tính năng này giúp các nhóm giảm lỗi trong môi trường sản phẩm.

Nhiều công ty xây dựng ứng dụng web đã chuyển đổi mã nguồn sang TypeScript. Nếu xu hướng này tiếp tục, TypeScript có thể thống trị lĩnh vực phát triển web trong vài năm tới.

## 📝 Phân tích ngữ pháp

### Câu 1: "TypeScript, which is a superset of JavaScript, has become one of the most popular languages on GitHub."
- **Cấu trúc:** S + , which + V + , + has become + O
- **Ngữ pháp:** Mệnh đề quan hệ không xác định (Non-defining relative clause) với "which" — bổ sung thông tin về TypeScript
- **Ví dụ tương tự:** Python, which is easy to learn, is used in many AI projects.

### Câu 2: "Developers who work on large projects prefer TypeScript because it offers static typing."
- **Cấu trúc:** S + who + V + V + O + because + S + V + O
- **Ngữ pháp:** Mệnh đề quan hệ xác định (Defining relative clause) với "who" — xác định nhóm developer nào
- **Ví dụ tương tự:** Engineers who understand cloud computing earn higher salaries.

### Câu 3: "If a developer makes a type error, the compiler will catch it before the code runs."
- **Cấu trúc:** If + S + V(present), S + will + V
- **Ngữ pháp:** Câu điều kiện loại 1 (First conditional) — điều kiện có thể xảy ra trong thực tế
- **Ví dụ tương tự:** If you write clean code, the team will review it faster.

### Câu 4: "Many companies that build web applications have already migrated their codebases to TypeScript."
- **Cấu trúc:** S + that + V + have already + V(past participle) + O
- **Ngữ pháp:** Mệnh đề quan hệ xác định với "that" + thì hiện tại hoàn thành (Present Perfect) với "already"
- **Ví dụ tương tự:** Teams that use agile methods have already improved their delivery speed.

## ✏️ Bài tập

### Comprehension (Đọc hiểu)
1. TypeScript xếp hạng cao hơn JavaScript ở tiêu chí nào trên GitHub?
2. Tại sao developer thích dùng TypeScript cho dự án lớn?
3. Điều gì có thể xảy ra nếu xu hướng TypeScript tiếp tục?

### Vocabulary (Từ vựng)
Điền từ thích hợp:
1. The ___ will catch errors before the code runs.
2. Our team has ___ the old system to a new framework.
3. TypeScript is a ___ of JavaScript.
4. This language ___ number one on GitHub.

**Comprehension:**
1. Số repositories mới được tạo trong năm — TypeScript có nhiều repo mới hơn JavaScript.
2. Vì TypeScript cung cấp static typing, giúp phát hiện lỗi kiểu dữ liệu trước khi chạy code, giảm bug trong production.
3. TypeScript có thể thống trị (dominate) lĩnh vực phát triển web trong vài năm tới.
