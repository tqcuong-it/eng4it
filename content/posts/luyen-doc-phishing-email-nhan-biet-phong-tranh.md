> 📖 **Cấp độ:** Pre-Intermediate
> ⏱️ **Thời gian đọc:** ~4 phút
> 📰 **Chủ đề:** Cybersecurity / Email Scam

## 📰 Bài đọc (English)

A new report from a cybersecurity firm, which {{< vocab w="tracks" ipa="/træks/" vn="theo dõi" t="verb" >}} online threats, shows that {{< vocab w="phishing" ipa="/ˈfɪʃɪŋ/" vn="lừa đảo trực tuyến" t="noun" >}} emails increased by 40% in the first half of 2024. These {{< vocab w="scam" ipa="/skæm/" vn="lừa đảo" t="noun" >}} emails, which often look like {{< vocab w="legitimate" ipa="/lɪˈdʒɪtɪmət/" vn="hợp pháp, chính thống" t="adj" >}} messages from banks or tech companies, try to steal personal {{< vocab w="credentials" ipa="/krɪˈdenʃəlz/" vn="thông tin đăng nhập" t="noun" >}}.

Security experts say that if you receive an email that asks for your password, you should never click the link. {{< vocab w="Attackers" ipa="/əˈtækərz/" vn="kẻ tấn công" t="noun" >}} create {{< vocab w="fake" ipa="/feɪk/" vn="giả mạo" t="adj" >}} websites that look exactly like real ones. If an employee clicks on a phishing link, the entire company network could be {{< vocab w="compromised" ipa="/ˈkɑːmprəmaɪzd/" vn="bị xâm nhập" t="verb" >}}.

Companies which take security seriously are now using {{< vocab w="multi-factor" ipa="/ˌmʌltiˈfæktər/" vn="đa yếu tố" t="adj" >}} authentication (MFA) to protect their systems. Experts {{< vocab w="recommend" ipa="/ˌrekəˈmend/" vn="khuyến nghị" t="verb" >}} that if you are unsure about an email, you should {{< vocab w="verify" ipa="/ˈverɪfaɪ/" vn="xác minh" t="verb" >}} it by contacting the sender directly.

## 🇻🇳 Bản dịch tiếng Việt

Một báo cáo mới từ công ty an ninh mạng chuyên theo dõi các mối đe dọa trực tuyến cho thấy email lừa đảo (phishing) đã tăng 40% trong nửa đầu năm 2024. Những email lừa đảo này, thường trông giống như tin nhắn chính thống từ ngân hàng hoặc công ty công nghệ, cố gắng đánh cắp thông tin đăng nhập cá nhân.

Các chuyên gia bảo mật cho biết nếu bạn nhận được email yêu cầu mật khẩu, bạn không bao giờ nên nhấp vào liên kết. Kẻ tấn công tạo ra các trang web giả mạo trông y hệt trang thật. Nếu một nhân viên nhấp vào liên kết phishing, toàn bộ mạng lưới công ty có thể bị xâm nhập.

Các công ty coi trọng bảo mật hiện đang sử dụng xác thực đa yếu tố (MFA) để bảo vệ hệ thống. Chuyên gia khuyến nghị rằng nếu bạn không chắc chắn về một email, bạn nên xác minh bằng cách liên hệ trực tiếp với người gửi.

## 📝 Phân tích ngữ pháp

### Câu 1: "These scam emails, which often look like legitimate messages from banks, try to steal personal credentials."
- **Cấu trúc:** S + , which + V + , + V + O
- **Ngữ pháp:** Mệnh đề quan hệ không xác định (Non-defining relative clause) — bổ sung mô tả cho "scam emails"
- **Ví dụ tương tự:** These apps, which are free to download, collect user data secretly.

### Câu 2: "If you receive an email that asks for your password, you should never click the link."
- **Cấu trúc:** If + S + V(present) + that + V, S + should + never + V
- **Ngữ pháp:** Câu điều kiện loại 1 kết hợp mệnh đề quan hệ xác định (defining relative clause) với "that"
- **Ví dụ tương tự:** If you find a bug that crashes the app, you should report it immediately.

### Câu 3: "If an employee clicks on a phishing link, the entire company network could be compromised."
- **Cấu trúc:** If + S + V(present), S + could + be + V(pp)
- **Ngữ pháp:** Câu điều kiện loại 1 với "could" + bị động — diễn tả hậu quả có thể xảy ra
- **Ví dụ tương tự:** If a hacker gains access, the database could be destroyed.

### Câu 4: "Companies which take security seriously are now using multi-factor authentication."
- **Cấu trúc:** S + which + V + adv + are + V-ing + O
- **Ngữ pháp:** Mệnh đề quan hệ xác định (Defining relative clause) — xác định "companies" nào
- **Ví dụ tương tự:** Developers who write clean code are valued by every team.

## ✏️ Bài tập

### Comprehension (Đọc hiểu)
1. By how much did phishing emails increase in 2024?
2. What do phishing emails try to steal?
3. What do experts recommend if you are unsure about an email?

### Vocabulary (Từ vựng)
Điền từ thích hợp:
1. Hackers created a ___ website to trick users into entering their passwords.
2. The security team ___ all online threats 24 hours a day.
3. You should ___ the sender's identity before opening any attachment.
4. The company's server was ___ after an employee clicked a malicious link.
