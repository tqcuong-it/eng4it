## 📋 Phương pháp Cornell là gì?

### Giới thiệu

Phương pháp **Cornell Note-taking** được phát triển tại Đại học Cornell (Mỹ). Đây là cách ghi chú **có hệ thống** giúp bạn vừa nghe vừa ghi hiệu quả — rất phù hợp cho IELTS Listening Section 3 & 4.

### Cấu trúc trang ghi chú Cornell

```
┌─────────────────────────────────────────┐
│              TIÊU ĐỀ / CHỦ ĐỀ          │
├──────────┬──────────────────────────────┤
│          │                              │
│  CỘT    │       GHI CHÚ CHÍNH          │
│  TỪ KHÓA│                              │
│          │  - Ý chính 1                 │
│  (Viết   │    + chi tiết                │
│   sau    │    + ví dụ                   │
│   khi    │                              │
│   nghe)  │  - Ý chính 2                 │
│          │    + chi tiết                 │
│          │    + số liệu                 │
│          │                              │
├──────────┴──────────────────────────────┤
│              TÓM TẮT                    │
│  (Viết sau khi nghe xong — 2-3 câu)    │
└─────────────────────────────────────────┘
```

### Cách áp dụng cho IELTS Listening

| Bước | Khi nào | Làm gì |
|------|---------|--------|
| 1 | Trước khi nghe | Viết tiêu đề + đọc câu hỏi |
| 2 | Trong khi nghe | Ghi nhanh vào cột GHI CHÚ CHÍNH |
| 3 | Sau khi nghe | Viết từ khóa vào cột trái |
| 4 | Ôn tập | Viết tóm tắt ở phần dưới |

## 📝 Hệ thống viết tắt cho IELTS Listening

### Viết tắt cơ bản

| Viết tắt | Nghĩa | Ví dụ |
|----------|-------|-------|
| → | dẫn đến, kết quả | pollution → health problems |
| ← | nguyên nhân | ← deforestation |
| ↑ | tăng, nhiều hơn | population ↑ |
| ↓ | giảm, ít hơn | cost ↓ |
| = | bằng, tương đương | A = B |
| ≠ | khác, không bằng | A ≠ B |
| ∴ | vì vậy, cho nên | ∴ need more research |
| ∵ | bởi vì | ∵ lack of funding |
| ~ | khoảng, xấp xỉ | ~50% |
| > | lớn hơn | income > expenses |
| < | nhỏ hơn | time < expected |

### Viết tắt từ vựng thường dùng

| Viết tắt | Từ đầy đủ |
|----------|-----------|
| govt | government |
| env | environment |
| dev | development |
| edu | education |
| tech | technology |
| econ | economy |
| pop | population |
| info | information |
| esp | especially |
| diff | different/difference |
| imp | important |
| prob | problem |

### Ký hiệu nhanh

| Ký hiệu | Nghĩa |
|----------|-------|
| ★ | quan trọng |
| ? | không chắc chắn, cần kiểm tra |
| ! | ngạc nhiên, đáng chú ý |
| ✓ | đã xác nhận |
| ✗ | sai, loại bỏ |

## 🎧 Bài tập 1: Cornell Note-taking — Section 3

👉 [British Council — B2 Listening](https://learnenglish.britishcouncil.org/skills/listening/b2-listening)

**Hướng dẫn:**
1. Chuẩn bị giấy chia 2 cột theo mẫu Cornell
2. Nghe bài nghe Section 3
3. Ghi chú vào cột phải (dùng viết tắt!)
4. Sau khi nghe, điền từ khóa vào cột trái
5. Viết 2-3 câu tóm tắt ở dưới

---

## 🎧 Bài tập 2: Cornell Note-taking — Section 4

👉 [IELTS Liz — Section 4 Lecture](https://ieltsliz.com/ielts-listening-section-4/)

**Hướng dẫn:**
1. Nghe bài giảng Section 4
2. Ghi chú theo phương pháp Cornell
3. So sánh ghi chú của bạn với transcript
4. Đánh giá: bạn bắt được bao nhiêu % ý chính?

**Mục tiêu:** Ghi được ≥ 70% ý chính

---

## 🎧 Bài tập 3: Dictation với viết tắt

👉 [BBC 6 Minute English](https://www.bbc.co.uk/learningenglish/english/features/6-minute-english)

**Cách làm:**
1. Mở BBC 6 Minute English
2. Nghe từng đoạn 30 giây
3. Ghi lại **bằng viết tắt** (không viết đầy đủ!)
4. Sau khi nghe xong, "dịch" viết tắt thành câu đầy đủ
5. So sánh với transcript

**Ví dụ:**

Audio: "The government has invested significantly in renewable energy development"

Ghi tắt: `govt → invest ↑ in renew energy dev`

---

## 📝 So sánh: Ghi chú tốt vs Ghi chú kém

### ❌ Ghi chú kém

```
The speaker talked about the environment
and said that pollution is bad and we need
to do something about it because it affects
health and the economy...
```

→ Quá dài, không kịp nghe tiếp!

### ✅ Ghi chú tốt

```
env → pollution
  - health ↓
  - econ ↓
  ∴ need action
```

→ Ngắn gọn, vẫn đủ ý, kịp nghe câu tiếp!

---

## 💡 Mẹo ghi chú trong phòng thi

1. **Dùng bút chì** — dễ xóa sửa
2. **Viết BÊN CẠNH câu hỏi** — không cần giấy riêng
3. **Chỉ ghi từ khóa** — không ghi nguyên câu
4. **Dùng mũi tên** thay vì viết "leads to", "results in"
5. **Đánh dấu ? cho câu không chắc** — quay lại sau

> 💡 **Nhớ:** Ghi chú là công cụ hỗ trợ, không phải mục đích. Đừng mải ghi mà quên nghe! 📝
