## 📋 Tại sao Error Analysis quan trọng?

Nhiều bạn làm test xong chỉ xem điểm rồi bỏ qua. Đây là sai lầm lớn nhất!

**Error Analysis** giúp bạn:
- 🎯 Biết chính xác **loại lỗi** nào bạn hay mắc
- 📊 Tìm ra **pattern** — lỗi lặp đi lặp lại
- 📈 Lập kế hoạch **cải thiện có mục tiêu**
- ⏰ Tiết kiệm thời gian — chỉ tập trung vào điểm yếu

## 🔍 Bước 1: Phân loại lỗi sai

|-------|---------|-------------|----------------|----------|
| | | | | |
| | | | | |

### Các loại lỗi thường gặp

| Mã | Loại lỗi | Mô tả | Ví dụ |
|----|----------|--------|-------|
| SP | Spelling | Viết sai chính tả | "libary" thay vì "library" |
| NUM | Number/Date | Nghe nhầm số | 15 thay vì 50 |
| DIS | Distractor | Chọn đáp án bẫy | Đáp án thay đổi giữa chừng |
| MIS | Miss | Không nghe kịp | Mất tập trung, tốc độ nhanh |
| WL | Word Limit | Vượt số từ cho phép | Viết 3 từ khi đề yêu cầu ≤2 |
| FRM | Form | Sai dạng từ | "travel" thay vì "travelling" |
| VOC | Vocabulary | Không hiểu từ | Từ academic không biết |
| PAR | Paraphrase | Không nhận ra paraphrase | Audio nói khác đề |

## 📝 Bước 4: Lập Improvement Plan

### Template kế hoạch cải thiện

```
📋 IMPROVEMENT PLAN — TUẦN 26
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 MỤC TIÊU:
Band hiện tại: ___
Band mục tiêu: ___
Số câu cần cải thiện: ___

🔴 ƯU TIÊN 1 (Lỗi nghiêm trọng nhất):
Loại lỗi: _______________
Số lần mắc: ___
Kế hoạch: _______________
Thời gian luyện: ___ phút/ngày

🟡 ƯU TIÊN 2:
Loại lỗi: _______________
Kế hoạch: _______________
Thời gian luyện: ___ phút/ngày

🟢 DUY TRÌ:
Điểm mạnh: _______________
Cách duy trì: _______________
```

### Gợi ý cải thiện theo loại lỗi

| Loại lỗi | Cách cải thiện |
|----------|----------------|
| Spelling | Luyện dictation 15 phút/ngày, học spelling rules |
| Number | Nghe BBC news, tập viết số khi nghe |
| Distractor | Học signpost words: "but", "actually", "I mean" |
| Miss | Luyện speed listening 1.25x |
| Word Limit | Luôn đọc kỹ "NO MORE THAN X WORDS" trước |
| Form | Ôn lại grammar: singular/plural, verb forms |
| Vocabulary | Học 10 từ academic mới mỗi ngày |
| Paraphrase | Luyện synonym matching |
