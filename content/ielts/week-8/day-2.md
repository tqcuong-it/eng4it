## 🌙 Buổi tối (40-60 phút) — Grammar: Zero Conditional

### 📖 Zero Conditional là gì?

**Zero Conditional** dùng để nói về **sự thật luôn đúng**, **quy luật tự nhiên**, hoặc **kết quả chắc chắn xảy ra**.

### 🔑 Cấu trúc

| If clause | Main clause |
|-----------|-------------|
| If + S + **V (present simple)** | S + **V (present simple)** |

> 📌 **Ví dụ:**
> - If you **heat** water to 100°C, it **boils**. (Nếu bạn đun nước đến 100°C, nó sôi.)
> - If it **rains**, the ground **gets** wet. (Nếu trời mưa, mặt đất ướt.)
> - If you **mix** red and blue, you **get** purple. (Nếu bạn pha đỏ và xanh, bạn được tím.)

### Đặc điểm Zero Conditional

| Đặc điểm | Giải thích |
|-----------|------------|
| Cả hai vế đều **present simple** | Vì nói về sự thật, không phải tương lai |
| Có thể thay **if** bằng **when** | If/When you heat water, it boils. |
| Kết quả **luôn luôn** xảy ra | 100% đúng, không phải "có thể" |

> 🧠 **Mẹo nhớ:** Zero Conditional = **Sự thật hiển nhiên**. If + present → present. Đơn giản nhất trong các conditional!

### ✏️ Bài tập 1: Nối hai vế

Nối vế trái với vế phải phù hợp:

| # | If clause | | Main clause |
|---|-----------|---|-------------|
| 1 | If you pollute rivers, | | a. ice melts. |
| 2 | If the temperature rises, | | b. plants die. |
| 3 | If there is no rain, | | c. fish die. |
| 4 | If you burn fossil fuels, | | d. the air gets dirty. |
| 5 | If you cut down trees, | | e. animals lose their habitat. |

### ✏️ Bài tập 2: Hoàn thành câu

Dùng Zero Conditional để hoàn thành câu:

1. If you ______ (not recycle), you ______ (create) more waste.
2. If factories ______ (dump) chemicals, the soil ______ (become) toxic.
3. If the ozone layer ______ (get) thinner, UV radiation ______ (increase).
4. If people ______ (litter), the environment ______ (suffer).
5. If you ______ (burn) plastic, it ______ (release) toxic smoke.

### ✏️ Bài tập 3: Viết câu Zero Conditional

Viết câu Zero Conditional từ gợi ý:

1. (cut trees / soil erosion)
2. (pollute air / people get sick)
3. (recycle paper / save trees)
4. (use fossil fuels / produce carbon)
5. (protect forests / animals survive)

### ✅ Đáp án Bài tập 1

| Câu | Đáp án | Giải thích |
|-----|--------|------------|
| 1 | c | Ô nhiễm sông → cá chết |
| 2 | a | Nhiệt độ tăng → băng tan |
| 3 | b | Không mưa → cây chết |
| 4 | d | Đốt nhiên liệu hóa thạch → không khí bẩn |
| 5 | e | Chặt cây → động vật mất nơi ở |

### ✅ Đáp án Bài tập 2

| Câu | Đáp án | Giải thích |
|-----|--------|------------|
| 1 | don't recycle / create | Phủ định: don't + V nguyên mẫu |
| 2 | dump / becomes | factories (số nhiều) → dump; soil (số ít) → becomes |
| 3 | gets / increases | Cả hai vế present simple |
| 4 | litter / suffers | people → litter; environment → suffers |
| 5 | burn / releases | Cả hai vế present simple |

### ✅ Đáp án Bài tập 3

| Câu | Đáp án |
|-----|--------|
| 1 | If you cut trees, soil erosion happens. |
| 2 | If you pollute the air, people get sick. |
| 3 | If you recycle paper, you save trees. |
| 4 | If you use fossil fuels, you produce carbon. |
| 5 | If you protect forests, animals survive. |

## 😴 Trước ngủ (10-15 phút) — Nghe + Đọc

### 🎧 Listening: Nghe BBC (5 phút)

👉 [BBC Learning English - What's the problem with plastic?](https://www.bbc.co.uk/learningenglish/english/features/6-minute-english/2023-0316)

> 💡 **Mẹo:** Nghe lần 1 không nhìn text. Nghe lần 2 vừa nghe vừa đọc transcript.

### 📖 Reading: Đoạn đọc ngắn (5 phút)

> **Pollution** is one of the biggest problems in the world today. If factories **dump** **toxic** **chemicals** into rivers, the water becomes **contaminated**. If people burn **fossil fuels**, it produces **carbon** **emissions** and causes **global warming**.
>
> **Plastic** **waste** is also a serious issue. If we don't **recycle**, **plastic** stays in the **soil** for hundreds of years. **Deforestation** makes the problem worse. If we cut down trees, the air gets dirtier. We need to act now!

*(85 từ)*

### ❓ Câu hỏi đọc hiểu

1. What happens if factories dump chemicals into rivers?
2. What do fossil fuels produce when burned?
3. How long does plastic stay in the soil?
4. What is deforestation?
5. What happens if we cut down trees?

| Câu | Đáp án |
|-----|--------|
| 1 | The water becomes contaminated. |
| 2 | They produce carbon emissions. |
| 3 | Plastic stays in the soil for hundreds of years. |
| 4 | Deforestation is cutting down trees / destroying forests. |
| 5 | The air gets dirtier. |
