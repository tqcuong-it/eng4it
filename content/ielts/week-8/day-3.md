## 🌙 Buổi tối (40-60 phút) — Grammar: First Conditional (Ôn tập)

### 📖 First Conditional là gì?

**First Conditional** dùng để nói về **điều có thể xảy ra trong tương lai** — khả năng thực tế, có thể xảy ra.

### 🔑 Cấu trúc

| If clause | Main clause |
|-----------|-------------|
| If + S + **V (present simple)** | S + **will + V (nguyên mẫu)** |

> 📌 **Ví dụ:**
> - If it **rains** tomorrow, I **will stay** home. (Nếu ngày mai mưa, tôi sẽ ở nhà.)
> - If the temperature **rises**, ice caps **will melt**. (Nếu nhiệt độ tăng, băng sẽ tan.)
> - If we **don't act** now, climate change **will get** worse. (Nếu chúng ta không hành động, BĐKH sẽ tệ hơn.)

### So sánh Zero vs First Conditional

| | Zero Conditional | First Conditional |
|---|---|---|
| Cấu trúc | If + present, present | If + present, **will** + V |
| Ý nghĩa | Sự thật luôn đúng (100%) | Có thể xảy ra trong tương lai |
| Ví dụ | If you heat ice, it melts. | If it rains tomorrow, I will stay home. |

> 🧠 **Mẹo nhớ:** Zero = sự thật hiển nhiên (luôn luôn). First = tình huống có thể xảy ra (tương lai).

### ✏️ Bài tập 1: Zero hay First Conditional?

Chọn loại conditional phù hợp và hoàn thành câu:

1. If you heat water, it ______ (boil). → Zero / First?
2. If the storm ______ (come), we ______ (cancel) the trip. → Zero / First?
3. If it ______ (rain), the ground ______ (get) wet. → Zero / First?
4. If the government ______ (not act), pollution ______ (increase). → Zero / First?
5. If you freeze water, it ______ (become) ice. → Zero / First?

### ✏️ Bài tập 2: Hoàn thành câu First Conditional

1. If the temperature ______ (rise), sea levels ______ (increase).
2. If we ______ (plant) more trees, the air ______ (be) cleaner.
3. If there ______ (be) a drought, crops ______ (die).
4. If the weather ______ (be) severe, schools ______ (close).
5. If we ______ (not reduce) emissions, the climate ______ (change) dramatically.

### ✏️ Bài tập 3: Viết câu First Conditional

Viết câu First Conditional từ gợi ý:

1. (heatwave / come → people / suffer)
2. (we / save energy → reduce / emissions)
3. (ice caps / melt → sea level / rise)
4. (government / invest in solar energy → air / be cleaner)
5. (deforestation / continue → many species / disappear)

---


### ✅ Đáp án Bài tập 1

| Câu | Đáp án | Giải thích |
|-----|--------|------------|
| 1 | boils → **Zero** | Sự thật khoa học luôn đúng |
| 2 | comes / will cancel → **First** | Tình huống có thể xảy ra |
| 3 | rains / gets → **Zero** | Quy luật tự nhiên |
| 4 | doesn't act / will increase → **First** | Dự đoán tương lai |
| 5 | becomes → **Zero** | Sự thật khoa học |

### ✅ Đáp án Bài tập 2

| Câu | Đáp án | Giải thích |
|-----|--------|------------|
| 1 | rises / will increase | If + present, will + V |
| 2 | plant / will be | If + present, will + V |
| 3 | is / will die | If + present, will + V |
| 4 | is / will close | If + present, will + V |
| 5 | don't reduce / will change | If + present (phủ định), will + V |

### ✅ Đáp án Bài tập 3

| Câu | Đáp án |
|-----|--------|
| 1 | If a heatwave comes, people will suffer. |
| 2 | If we save energy, we will reduce emissions. |
| 3 | If ice caps melt, sea levels will rise. |
| 4 | If the government invests in solar energy, the air will be cleaner. |
| 5 | If deforestation continues, many species will disappear. |

## 😴 Trước ngủ (10-15 phút) — Nghe + Đọc

### 🎧 Listening: Nghe BBC (5 phút)

👉 [BBC Learning English - Is the weather getting worse?](https://www.bbc.co.uk/learningenglish/english/features/6-minute-english/2023-0727)

> 💡 **Mẹo:** Nghe lần 1 không nhìn text. Nghe lần 2 vừa nghe vừa đọc transcript.

### 📖 Reading: Đoạn đọc ngắn (5 phút)

> **Climate change** is a serious problem. The **temperature** is rising every year. If it continues, **ice caps** will melt and **floods** will become more common. Scientists say the **weather** will be more **severe** in the future.
>
> **Heatwaves** and **droughts** will affect many countries. If we don't reduce **carbon** **emissions**, the **atmosphere** will get warmer. The **forecast** is not good, but if we act now, we will still have a chance to save our planet.

*(80 từ)*

### ❓ Câu hỏi đọc hiểu

1. What is happening to the temperature?
2. What will happen if ice caps melt?
3. What will the weather be like in the future?
4. What do we need to reduce?
5. Is there still hope according to the passage?

| 2 | Floods will become more common. |
