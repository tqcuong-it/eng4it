## 🎧 Bài tập ôn tập tổng hợp

### Bài tập 1: Quick Review Quiz

Trả lời nhanh:

1. Section 1 là hội thoại hay độc thoại? → __________
2. Section 2 là hội thoại hay độc thoại? → __________
3. "Double seven" = bao nhiêu? → __________
4. "Fifteen" nhấn âm đầu hay cuối? → __________
5. "First floor" trong British English = tầng mấy? → __________
6. Hyphenated word (ví dụ "well-known") tính là mấy từ? → __________
7. Signpost language là gì? → __________
8. Khi miss 1 câu, nên làm gì? → __________

### Bài tập 2: Dạng câu hỏi identification

Xác định dạng câu hỏi:

| Mô tả | Dạng |
|-------|------|
| 1. Điền thông tin vào biểu mẫu (tên, SĐT, địa chỉ) | |
| 2. Chọn A, B, hoặc C | |
| 3. Nối hoạt động với địa điểm | |
| 4. Phân loại thông tin vào 2-3 nhóm | |
| 5. Điền từ vào câu chưa hoàn chỉnh | |
| 6. Trả lời câu hỏi What/Where/When | |
| 7. Gắn nhãn vị trí trên bản đồ | |
| 8. Điền từ vào ghi chú dạng bullet point | |

## 💡 Kế hoạch tiếp theo

Bạn đã hoàn thành **Section 1 & 2** của IELTS Listening! Đây là 2 phần "dễ ăn điểm" nhất.

### Điểm mục tiêu tổng Listening

| Section | Mục tiêu | Bạn đạt |
|---------|---------|---------|
| Section 1 | 9-10/10 | ___/10 |
| Section 2 | 7-8/10 | ___/10 |
| Section 3 | 6-7/10 | (Chưa học) |
| Section 4 | 5-6/10 | (Chưa học) |
| **Tổng** | **27-31/40** → Band 6.0-7.0 | |

### Lời khuyên

1. **Nghe mỗi ngày** — ít nhất 15 phút tiếng Anh (podcast, BBC, TED)
2. **Ôn paraphrase** — đây là kỹ năng QUAN TRỌNG NHẤT
3. **Luyện tốc độ viết** — viết tắt khi ghi chú
4. **Đừng sợ sai** — mỗi lần sai là 1 lần học
