# Eng4IT — Tiếng Anh cho Dân IT

Blog học tiếng Anh miễn phí dành cho Developer, BrSE, PM tại Nhật Bản và Việt Nam.

🌐 **https://eng4it.com**

## Nội dung

- 📚 **IELTS** — Lộ trình tự học IELTS Academic 6.0 trong 10 tháng, giáo án chi tiết từng ngày
- 💼 **Workplace English** — Mẫu câu email, họp, giao tiếp công sở cho developer
- 🔤 **Tiếng Anh Căn Bản** — Phát âm IPA, ngữ pháp cơ bản, từ vựng nền tảng

## Tech Stack

- [Hugo](https://gohugo.io/) — Static site generator
- [PaperMod](https://github.com/adityatelange/hugo-PaperMod) — Theme
- [Cloudflare Pages](https://pages.cloudflare.com/) — Hosting (free)
- Font: [Be Vietnam Pro](https://fonts.google.com/specimen/Be+Vietnam+Pro) — Tối ưu cho tiếng Việt
