## 🌙 Buổi tối (40-60 phút) — Grammar: Second Conditional

### 📖 Second Conditional là gì?

**Second Conditional** dùng để nói về **tình huống không có thật / giả định ở hiện tại** hoặc **điều khó xảy ra**.

### 🔑 Cấu trúc

| If clause | Main clause |
|-----------|-------------|
| If + S + **V (past simple)** | S + **would + V (nguyên mẫu)** |

> 📌 **Ví dụ:**
> - If I **had** more money, I **would donate** to wildlife conservation. (Nếu tôi có nhiều tiền hơn, tôi sẽ quyên góp cho bảo tồn.)
> - If everyone **recycled**, the world **would be** cleaner. (Nếu mọi người tái chế, thế giới sẽ sạch hơn.)
> - If I **were** the president, I **would ban** deforestation. (Nếu tôi là tổng thống, tôi sẽ cấm phá rừng.)

### Lưu ý đặc biệt

| Điểm | Giải thích |
|------|------------|
| **were** cho tất cả ngôi | If I **were**... / If he **were**... (formal) |
| Dùng **was** cũng OK | If I **was**... (informal, speaking) |
| **would** = sẽ (giả định) | Khác "will" (thật), "would" = tưởng tượng |

### So sánh 3 loại Conditional

| | Zero | First | Second |
|---|---|---|---|
| Cấu trúc | If + present, present | If + present, will + V | If + past, would + V |
| Ý nghĩa | Sự thật | Có thể xảy ra | Giả định, khó xảy ra |
| Ví dụ | If ice melts, it becomes water. | If it rains, I will stay home. | If I were rich, I would travel. |

> 🧠 **Mẹo nhớ:** Second = Giả định / Mơ ước. Dùng **past** ở vế If nhưng KHÔNG nói về quá khứ — nói về tình huống **không có thật ở hiện tại**!

### ✏️ Bài tập 1: Hoàn thành câu Second Conditional

1. If I ______ (be) a scientist, I ______ (study) endangered species.
2. If the government ______ (invest) more, they ______ (save) the rainforest.
3. If people ______ (stop) littering, the ocean ______ (be) cleaner.
4. If we ______ (use) only renewable energy, pollution ______ (decrease).
5. If I ______ (live) near a forest, I ______ (volunteer) for conservation.

### ✏️ Bài tập 2: First hay Second Conditional?

1. If we plant more trees, the air ______ (be) cleaner. → First / Second?
2. If I were a millionaire, I ______ (fund) a national park. → First / Second?
3. If the storm comes tomorrow, we ______ (stay) inside. → First / Second?
4. If everyone stopped using plastic, the ocean ______ (be) healthier. → First / Second?
5. If you reduce waste, you ______ (help) the environment. → First / Second?

### ✏️ Bài tập 3: Viết câu Second Conditional

Viết câu Second Conditional từ gợi ý:

1. (I / be the mayor → ban / single-use plastic)
2. (companies / use solar energy → reduce / carbon emissions)
3. (people / care more → biodiversity / not decline)
4. (we / have more national parks → animals / be safer)
5. (I / can fly → explore / every ecosystem on Earth)

### ✅ Đáp án Bài tập 1

| Câu | Đáp án | Giải thích |
|-----|--------|------------|
| 1 | were / would study | If + past, would + V |
| 2 | invested / would save | If + past, would + V |
| 3 | stopped / would be | If + past, would + V |
| 4 | used / would decrease | If + past, would + V |
| 5 | lived / would volunteer | If + past, would + V |

### ✅ Đáp án Bài tập 2

| Câu | Đáp án | Giải thích |
|-----|--------|------------|
| 1 | will be → **First** | Khả năng thực tế |
| 2 | would fund → **Second** | Giả định (không phải triệu phú) |
| 3 | will stay → **First** | Có thể xảy ra |
| 4 | would be → **Second** | Giả định (mọi người chưa dừng) |
| 5 | will help → **First** | Khả năng thực tế |

### ✅ Đáp án Bài tập 3

| Câu | Đáp án |
|-----|--------|
| 1 | If I were the mayor, I would ban single-use plastic. |
| 2 | If companies used solar energy, they would reduce carbon emissions. |
| 3 | If people cared more, biodiversity would not decline. |
| 4 | If we had more national parks, animals would be safer. |
| 5 | If I could fly, I would explore every ecosystem on Earth. |

### 📖 Reading: Đoạn đọc ngắn (5 phút)

> **Conservation** is very important for our planet. If every country **protected** its forests, **biodiversity** would increase. If people **donated** more money, organizations would **restore** damaged **ecosystems**.
>
> Many **volunteers** around the world help with **campaigns** to raise **awareness**. **National parks** and nature **reserves** play a key role in keeping animals safe. If we **used** more **renewable** energy like **solar** power, we **would reduce** our **impact** on nature. Every small action matters!

*(85 từ)*

### ❓ Câu hỏi đọc hiểu

1. What would happen if every country protected its forests?
2. What do volunteers help with?
3. What role do national parks play?
4. What would happen if we used more renewable energy?
5. Does every small action matter?
