## 🌙 Buổi tối (40-60 phút) — Grammar: First Conditional (Câu điều kiện loại 1)

### 📖 First Conditional là gì?

**First Conditional** dùng để nói về điều **có thể xảy ra** trong tương lai, với một điều kiện:
- 📋 *If it **rains**, I **will** stay home.* (Nếu trời mưa, tôi sẽ ở nhà.)
- 💡 *If you **study** hard, you **will** pass.* (Nếu bạn học chăm, bạn sẽ đỗ.)

### 🔑 Công thức quan trọng nhất

| Phần | Công thức |
|------|-----------|
| **Mệnh đề IF** | If + S + V (Present Simple) |
| **Mệnh đề chính** | S + **will** + V (nguyên mẫu) |

| Ví dụ đầy đủ | Phân tích |
|--------------|-----------|
| **If** I **study** hard, I **will pass** the exam. | If + Present Simple → will + V |
| I **will pass** the exam **if** I **study** hard. | Đảo thứ tự cũng được (bỏ dấu phẩy) |

> 🧠 **Mẹo nhớ:** IF = **Present Simple** (KHÔNG dùng will). RESULT = **will + V**.
> - ✅ If it **rains**, I **will** stay home.
> - ❌ ~~If it **will rain**, I will stay home.~~

### Lưu ý quan trọng

- Mệnh đề IF có thể đứng **trước** hoặc **sau**:
  - If you come, I will be happy. *(dấu phẩy)*
  - I will be happy if you come. *(không dấu phẩy)*
- Có thể dùng **can/may/might** thay will:
  - If you study, you **can** pass. (có thể)
  - If it rains, we **might** cancel. (có lẽ)

### ✏️ Bài tập 1: Nối hai vế câu

Nối mệnh đề IF với mệnh đề chính phù hợp:

| Mệnh đề IF | Mệnh đề chính |
|------------|---------------|
| 1. If I study hard, | a. you will feel better. |
| 2. If it rains, | b. I will pass the exam. |
| 3. If you eat healthy food, | c. we will stay home. |
| 4. If she gets the job, | d. I will buy a new phone. |
| 5. If I save enough money, | e. she will be very happy. |

### ✏️ Bài tập 2: Hoàn thành câu

Chia động từ cho đúng:

1. If I ______ (have) time, I ______ (help) you.
2. If she ______ (study) hard, she ______ (pass) the exam.
3. If it ______ (rain) tomorrow, we ______ (not/go) to the park.
4. If you ______ (not/eat) breakfast, you ______ (be) hungry.
5. If they ______ (work) harder, they ______ (get) promoted.
6. If he ______ (learn) English, he ______ (find) a better job.
7. We ______ (be) late if we ______ (not/leave) now.
8. She ______ (not/come) if you ______ (not/invite) her.

### ✏️ Bài tập 3: Sửa lỗi sai

1. If it will rain, I will stay home.
2. If she studies hard, she pass the exam.
3. If I will have money, I will buy a car.
4. If you don't hurry, you will missing the train.
5. If he will come, we will be happy.
6. I will help you if you will ask me.

### ✅ Đáp án Bài tập 2

| Câu | Đáp án | Giải thích |
|-----|--------|------------|
| 1 | have / will help | If + Present Simple / will + V |
| 2 | studies / will pass | She → studies (thêm -s) / will + V |
| 3 | rains / won't go | If + Present Simple / won't + V |
| 4 | don't eat / will be | If + don't + V / will + V |
| 5 | work / will get | If + Present Simple / will + V |
| 6 | learns / will find | He → learns (thêm -s) / will + V |
| 7 | will be / don't leave | will + V / if + don't + V (mệnh đề IF ở sau) |
| 8 | won't come / don't invite | won't + V / if + don't + V |

### ✅ Đáp án Bài tập 3

| Câu | Lỗi sai | Sửa đúng | Giải thích |
|-----|---------|----------|------------|
| 1 | will rain | If it **rains** | Mệnh đề IF dùng Present Simple, không dùng will |
| 2 | pass | she **will pass** | Mệnh đề chính cần will |
| 3 | will have | If I **have** | Mệnh đề IF không dùng will |
| 4 | will missing | you will **miss** | Sau will → V nguyên mẫu (không phải V-ing) |
| 5 | will come | If he **comes** | Mệnh đề IF dùng Present Simple |
| 6 | will ask | if you **ask** me | Mệnh đề IF không dùng will |

### 📖 Reading: Đoạn đọc ngắn (5 phút)

Đọc đoạn văn sau — tất cả từ đều từ bài hôm nay:

> **Technology** is changing fast. If **robots** become smarter, they **will** do many jobs for us. **Artificial** intelligence is everywhere now. If you use the **internet**, you **will** see AI in many **apps**.
>
> **Smart** **devices** **connect** us to the **digital** world. If we **develop** new **software**, life **will** be easier. **Electric** cars **will** be popular if **energy** prices go down.
>
> If companies **innovate**, they **will** succeed. If you **update** your skills, you **will** find better jobs. The **future** belongs to people who **search** for knowledge **online**.

*(85 từ)*

### ❓ Câu hỏi (trả lời ngắn gọn)

1. **What** will robots do if they become smarter?
2. **Where** will you see AI?
3. **What** will happen if we develop new software?
4. **When** will electric cars be popular?
5. **What** will happen if you update your skills?
